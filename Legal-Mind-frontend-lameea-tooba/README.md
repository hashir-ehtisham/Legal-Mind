# ⚖️ Legal Mind Frontend

An AI-powered legal assistance platform that helps users understand Pakistani laws, receive AI-generated legal guidance, and discover lawyers based on their legal needs.

## 📖 Overview

Legal Mind is a modern legal technology platform built to simplify legal assistance in Pakistan. The frontend offers an intuitive and responsive user experience featuring AI consultations, lawyer discovery, legal guidelines, and secure user interactions.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/your-username/legal-mind-frontend.git
```

Move into the project folder

```bash
cd legal-mind-frontend
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

---

## 📦 Production Build

Build the application

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
